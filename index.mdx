<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">
    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with Ivy in minutes
  </p>

  <Steps>
    <Step title="Install Ivy">
      Clone the repository and build from source using Cargo:
      
      ```bash
      git clone https://github.com/gtr/ivy.git
      cd ivy
      cargo build --release
      ```
      
      The compiled binary will be at `target/release/ivy`.
    </Step>

    <Step title="Start the REPL">
      Run Ivy without arguments to start the interactive REPL:
      
      ```bash
      ./target/release/ivy
      ```
      
      You'll see the Ivy welcome message:
      
      ```
        _
       (_)_   ___   _
       | \ \ / / | | |
       | |\ V /| |_| |
       |_| \_/  \__, |
                |___/
      
      Ivy - the friendly functional programming language
      Type :help for commands, :q to quit
      ```
    </Step>

    <Step title="Write your first program">
      Try a simple expression in the REPL:
      
      ```ivy
      ivy> let double = fn(x) => x * 2;
      ivy> double(21)
      42
      ```
      
      <Note>
        Notice that you didn't need to write type annotations — Ivy inferred the types automatically using Hindley-Milner type inference.
      </Note>
    </Step>

    <Step title="Run a program file">
      Create a file called `hello.ivy`:
      
      ```ivy
      fn greet(name: String): String =>
        "Hello, " ++ name ++ "!";
      
      let main = __println(greet("World"));
      ```
      
      Run it with:
      
      ```bash
      ./target/release/ivy hello.ivy
      ```
    </Step>
  </Steps>
</div>
